Image Thumbnail Tag

Usage

Generates and caches a proportional resized thumbnail of the provided image. Tag returns the path of the cached thumbnail.

Markup

{{ helper:image_thumb image="path/to/image.jpg" width="100" height="100" }}

Parameters

Parameter Description
image=*Required Specifies the path of the source image in which the thumbnail will be generated from.
Example:. image="path/to/image.jpg"
Alternatively, you can pass a template tag/viariable as the image source. But in order for that to work the source= parameter must be set to true
Example: image=your_template_variable source="true"
width= The max width the generated thumbnail can be. width="250"
height= The max height the generated thumbnail can be. height="250"
crop= If set to "true" the image will be cropped from the center of the image to the provided width and height. crop="true"
source= If set to "true" allows the parsing of template variables as the image source to the image. source="true"
